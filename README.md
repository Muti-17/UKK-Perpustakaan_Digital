# 📚 Aplikasi Perpustakaan Digital - UKK 2026
Aplikasi berbasis web untuk manajemen peminjaman buku di perpustakaan sekolah secara efisien dan terorganisir.

## 📝 Deskripsi Proyek
Proyek ini dibuat untuk memenuhi tugas Uji Kompetensi Keahlian (UKK) di SMKN 1 Lemahsugih. Sistem ini mencakup manajemen data buku, anggota, hingga proses transaksi peminjaman dan pengembalian yang dilengkapi dengan fitur laporan.

## 🚀 Fitur Utama
* **Multi-user Login:** Admin, Petugas, dan Peminjam.
* **Manajemen Buku:** Tambah, edit, dan hapus data buku serta kategorinya.
* **Sistem Peminjaman:** Proses pinjam dan kembali buku secara otomatis.
* **Fitur Ulasan:** Peminjam dapat memberikan ulasan dan rating pada buku.
* **Laporan PDF:** Generate laporan aktivitas perpustakaan.

## 🛠️ Teknologi yang Digunakan
* **Bahasa:** PHP (Native)
* **Database:** MySQL
* **Frontend:** HTML, CSS, Bootstrap
* **Tools:** Laragon, Visual Studio Code

## ⚙️ Cara Instalasi
1. Clone repository ini.
2. Impor database `perpustakaan.sql` yang ada di folder database ke phpMyAdmin.
3. Pindahkan folder proyek ke `htdocs`.
4. Sesuaikan konfigurasi database di file `koneksi.php` atau `.env`.
5. Buka `localhost/nama-folder` di browser.
